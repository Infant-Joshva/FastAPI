##### **Path Parameter:**



What is path parameters: 

* if you type something on a browser for example if you type a word **Mail**, you get a link something like: https://www.google.com/search?q=mail.....
* This part is called **Path:**"https://www.google.com/search?q=" and this part is called **Parameters:** mail.....
* The after the word you searched is called a parameters



Code:



app = FastAPI()



@app.get("/items/{item\_id}")

def for\_path\_parameters(item\_id: int):

&nbsp;   return {"item\_id": item\_id}



* **Note: we must mentioned the** *parameters* <b>curly phrases, and the url won't load without giving the </b>*parameters* 
* <b>How call -> http://127.0.0.1:8000/items/item\_id/1</b>

##### **Query Parameters:**



* Query parameters is same like path parameters but here we should not mention any parameters on path **"@app.get**"



Sample Code:



emp = \[

&nbsp;   {"id": 1, "name": "Vijay"},

&nbsp;   {"id": 2, "name": "Ajith"},

&nbsp;   {"id": 3, "name": "Karthik"},

]



\# Query parameters

@app.get("/items/")

def for\_queryparameters(q: str): # Search with name

&nbsp;   for i in emp:

&nbsp;       if i\["name"] == q:

&nbsp;           return i
