##### Dependency Injection:



* Inject the already returned code in this new code block
* Where these technique is used: It will use for **Session , DB, Authentication - Authorization, Normal method, Classes.** All these can use repeatedly without writing new code.



##### Types:



* Function level Dependency
* Class Level Dependency
* Sublevel -> Like Inheritance
* Global Level Dependency
* Yield or Generator - It like a cleanup dependency, like it will work before the **Request** and after the **Request.**

Eg: Work Flow: First this dependency will work and our Req will work and the balance part of the dependency will work
